## 📋 使用步骤

1. **环境设置**
   ```bash
   npm install
   ```

2. **配置个人信息**
   - 编辑 `profile-config.json`
   - 修改个人信息、技能、项目等

3. **生成 README**
   ```bash
   npm run generate
   ```

4. **部署到 GitHub**
   - 创建与用户名同名的仓库
   - 复制生成的 README.md
   - 提交并推送
